  • ¿De qué trata este proyecto?

    Curso Python - CoderHouse

    Este es mi Proyecto Final en el curso de PYTHON brindado por CoderHouse. El objetivo del mismo es mostrar mis primeras líneas de código, entendiendo cómo administrar perfiles, crear/editar/eliminar formularios, relacionar tablas, manejo de imágenes, configuración de admin y registro de mensajes/comentarios.

    Para esta entrega, el proyecto incluye un página de Inicio donde se pueden promocionar los beneficios del Autoservio, datos de contacto y las promociones vigentes. Además, aquellos clientes que se creen un usuario y se loguen podrán solicitar las promociones directamente por la página mediante comentarios. Otra funcionalidad disponible para los usuarios es la de cargar sus pedidos de mercadería aclarando fecha y lugar de entrega, hacer seguimiento de sus pedidos y el estado actual.

    A futuro, este proyecto tiene pensado incorporar nuevas funcionalidades: administración de stock, actualización de precios, carrito de compras y listas de favoritos.
